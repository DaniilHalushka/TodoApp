# TodoApp
It's small todo app, where you can add different tasks and see how many of them you do. 
This is your little assistant in organizing your daily tasks.

## Used Technologies
- Android SDK
- Android Lifecycle
- Kotlin
- Jetpack Compose
- MVVM
- Clean Architecture
- Kotlin Coroutines + Flow
- Compose Navigation
- Dagger/Hilt
- Room
- BDUI (DivKit)
