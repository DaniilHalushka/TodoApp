# TodoApp
It's a small to-do app where you can add different tasks and track how many of them you complete.
This is your little assistant for organizing your daily tasks.

## Used Technologies
- Android SDK
- Android Lifecycle
- Kotlin
- Jetpack Compose
- MVVM
- Clean Architecture
- Kotlin Coroutines + Flow
- Compose Navigation
- Dagger/Hilt
- Room
- BDUI (DivKit)
